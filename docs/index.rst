Prompt Engineer Documentation
==============================

Prompt Engineer is a CLI-based developer workbench for rapid AI prompt engineering iteration. It's designed for building AI-enabled applications (not chat interfaces) where prompts are compiled into the application with variables interpolated at runtime.

Key Features
------------

* **Workspace-Centric**: Prompts live in your application's source code
* **Variable Interpolation**: Support for file and value-based variables
* **Multiple Providers**: OpenAI, Ollama, LM Studio, OpenRouter support
* **Real-time Testing**: Test prompts with immediate feedback
* **Configuration Management**: User and workspace-level configuration
